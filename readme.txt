≪はじめに≫
必要な値を入力すると、プリプレグの裁断寸法を自動で計算されます。

≪設定≫
パイプの直径	：プリプレグを巻くアルミパイプの外形を入力してください。
パイプの長さ	：作成する桁の長さを入力してください。
積層数		：プリプレグの積層数を入力してください。
プリプレグ厚さ	：プリプレグの厚さを入力してください。
重ね代		：0，90度層の重ね代を入力してください。
パイプ余裕寸法	：パイプの長さにプラスする寸法を入力してください。
45度の余裕寸法　：45，-45度層のプリプレグ長さにプラスする寸法を入力してください。
繊維方向	：0，90，45，-45　のいずれかを入力してください。

90度層分割幅　　：最適化骨くん連携のための90度層分割幅を設定してください。

≪ボタン説明≫
計算開始	：ボタンをクリックするとプリプレグの長さと厚さを計算します。
		　計算が終了すると、印刷を実行するかのメッセージボックスが表示されます。

印刷開始	：結果の印刷をします。
		　同時に「History」フォルダへ結果を出力します。

最適化骨くん連携：最適化骨くん用のJSONファイルを出力します。

≪計算方法≫
  0，90度層
	長さ = パイプの長さ +  パイプ余裕寸法
	幅　 = ( 直径 * PI ) + 重ね代
  
　45，-45度層
	長さ = ( 直径 * PI / ROOT2 ) + ((パイプの長さ + パイプ余裕寸法) * ROOT2 ) + 45度の余裕寸法
	幅　 = ( 直径 * PI ) / ROOT2

　※「直径」の計算式
	直径 = パイプの直径 + ( プリプレグ厚さ * 2 * 現在の積層数 )

≪ファイル構成≫
　プリプレグ裁断寸法プログラム.exe
　　ダブルクリックで起動できます。

　Historyフォルダ
　　このフォルダに計算履歴が残ります。

　JSONフォルダ
　　最適化骨くんが読み込むためのJSONファイルが残ります。

  プリプレグ裁断寸法プログラムフォルダ
　　このフォルダにアプリのプロジェクトファイルがあります。
　　必要ない場合は消して頂いて構いません。
　　プロジェクトを開くには「プリプレグ裁断寸法プログラム.sln」をダブルクリックしてください。
